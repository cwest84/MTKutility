MTKutility is a rewrite as changes to Android made the original app unusable. The original app was inspired by Bastiaan Naber's AndroidMTK app.  The development objective was to create an Android equivalent of the Windows app MtkDlut (http://4river.a.la9.jp/gps/file/MtkDLut.htm). 

The app's roots are in Keisuke Suzuki's AndroidUSBSerialMonitorLite and FTdriver.  It started as an AGPS data update to a USB connected Qstarz GPS logger.  The development environment is Android Studio, a Samsung SGHI717 with Android 4.1 and an ASUS_X00ID with Android 8.1.  The finished app includes the GPX conversion fragment file from Bastiaan Naber's AndroidMTK (Nov 2014 version).
